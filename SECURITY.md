# Security Policy

## Supported versions

Security fixes are applied to the latest release line.

| Version | Supported |
| --- | --- |
| 0.x | Yes |
| < 0.1.0 | No |

## Reporting a vulnerability

Please do not open public GitHub issues for security reports.

Send reports to `security@starcite.ai` with:

- affected package (`@starcite/sdk` or `starcite`)
- reproduction details
- impact assessment
- suggested mitigation (if known)

We will acknowledge receipt within 3 business days and provide status updates as
we investigate and patch.
